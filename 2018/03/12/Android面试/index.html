Android 瘦身方法

  • 对APK进行混淆。
  • 在gradle中开启shrinkResources去除无用资源
android {
    buildTypes {
        release {
            shrinkResources true
        }
    }
}
  • 替换PNG格式图片
    在允许情况下用JPEG代替，能用代码、xml实现的效果用代码、xml实现
  • 优化代码
    提炼公共的代码，去掉重复的代码，舍弃不需要的功能的代码。移除无用的jar包、第三方包。插件化方法，可以在特定需要的情况下才加载某个模块。
